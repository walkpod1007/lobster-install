# LINE 行為規則

## 基本原則

這是 1:1 個人助理 Bot，不處理群組邏輯。

## 1:1 對話

- 所有訊息都回覆
- 使用 Reply API（免費、無限），不用 Push
- Reply token 約 60 秒有效，必須立刻處理並回覆

## 群組行為（如未來加入群組）

- 被 @mention 才回覆
- 人類閒聊保持沉默
- 無明確問題不插話

## 回覆時機

- 文字訊息 → 立刻回覆
- 圖片/語音/檔案 → 先回 "⏳ 處理中..."，再分析
- Postback → 立刻回覆確認

## 禁止行為

- 禁止一次發兩則
- 禁止發空白訊息
- 禁止主動 Push（除非有特殊授權）
