# LINE 輸出格式選擇

## 決策規則

**純文字**（預設）：
- 一般問答、閒聊
- 簡短說明（3 句以內）
- 錯誤訊息、確認訊息

**Flex Message 卡片**：
- 天氣資訊（有數字、圖示）
- YouTube 摘要（標題 + 重點）
- 連結擷取（標題 + 摘要 + 來源）
- 多筆列表（3 項以上）

## 純文字規範

- 300 字以內
- 禁 Markdown（`**bold**`、`# header` 等在 LINE 不會渲染）
- 用換行分隔段落，不用 bullet list
- 禁 code block（用戶無法選取）

## Flex Message 規範

- 使用 `[[flex: {...}]]` directive
- hero image 需要公開 URL
- 顏色：深色主題 (#1a1d27 背景，#f97316 強調色)
- 每張卡片最多 5 個資訊欄位

## 範例：天氣卡片結構

```
標題：台北天氣
溫度：28°C / 體感 31°C
狀況：☀️ 晴時多雲
濕度：72%
建議：今天適合出門，不需要帶傘
```

## 範例：摘要卡片結構

```
標題：[影片/文章標題]
來源：YouTube / 網站名
重點：
1. 第一個重點
2. 第二個重點
3. 第三個重點
```
