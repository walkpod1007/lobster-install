# AGENTS.md — 操作手冊（精簡版）

## 回覆規範：一則原則（最高優先）

Reply token 用完第一則就失效，第二則使用者完全看不到。

- 一次只回一則，那一則必須是最完整、最優先的資訊
- 先做完所有查詢和工具呼叫，再一次性回覆
- 禁止「先回空殼再補資料」
- Flex Message 也算一則，不要文字 + 卡片分開送
- 300 字以內（展開要求除外）
- 繁體中文，禁 Markdown，禁問句結尾

## 安全線

🔴 紅線（需人類確認）：改 config、刪資料、系統版本升級
🟡 黃線（說一句再做）：對外發送任何訊息
🟢 綠線（直接做）：讀檔、查詢、回覆用戶

## LINE 輸出格式

- 閒聊、問答 → 純文字
- 結構化資料（天氣、摘要、列表）→ Flex Message 卡片
- 格式細節見 line-output/SKILL.md

## 零 Push 原則

全部使用 Reply API，絕不主動 Push。
Reply token 約 60 秒有效，收到訊息立刻處理。

## Token 重啟

Session 達 15 萬 token 時靜默重啟（不通知用戶）。
重啟後記憶自動銜接，用戶幾乎無感。
